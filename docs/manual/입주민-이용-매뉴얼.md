## 1. 서비스 소개

**단지카**는 우리 아파트 **입주민만** 이용할 수 있는 공유·렌트 차량 서비스입니다.

| 대여 유형 | 설명 | 최대 기간 |
|-----------|------|-----------|
| 카셰어링(시간) | 시간 단위 이용 | 23시간 |
| 일 렌트 | 하루 단위 | 29일 |
| 월 렌트 | 월 단위 | 11개월 |

**이용 자격:** 만 26세 이상 · 운전면허 취득 1년 이상 · 입주민 인증·관리자 승인 · 면허 심사 통과 · 결제카드 등록

## 4. 차량 예약하기

### 4-1. 예약 화면 진입

- **홈** → 날짜·시간 선택 후 예약
- 또는 **마이페이지** → 예약하기

### 4-2. 일정·차량 선택

1. **시작·종료** 일시 선택 (시간 / 일 / 월 모드)
2. 이용 가능 **차량 목록** 확인 (요금·차종·연료)
3. 차량 카드에서 **요금 상세** 확인 후 선택

## 7. 대여 시작 (출고)

### 7-1. 대여 가능 시각

- 예약 **시작 30분 전**부터「대여하기」버튼 활성화

### 7-2. 출고 절차

1. **내 예약** → 해당 예약 → **대여하기**
2. 차량 **외관·내부 사진** 촬영
3. **주행거리·연료(배터리)** 입력
4. 확인 후 대여 시작 → 스마트키 이용 가능

## 10. 반납하기

### 10-1. 반납 절차

1. **지정 주차 구역**에 주차
2. **반납하기** 진입 → 차량 상태 **사진** 촬영
3. 최종 **주행거리·연료(배터리)** 입력
4. 반납 완료 → 이용 종료·포인트 적립

### 10-2. 주의사항

- 반납 **지연** 시 기존 요금 **2배** 패널티
- EV 배터리 20% 미만 반납 시 추가 비용 발생 가능

*반납 화면 (사진·최종 입력)*

## 14. 자주 묻는 질문 요약

**Q. 예약은 어떻게 하나요?**  
홈 또는 예약 메뉴에서 날짜·차량 선택 후 결제하면 확정됩니다.

**Q. 취소·환불은?**  
대여 시작 전까지 취소 가능. 카셰어링은 출고 1시간 기준, 일·월은 72시간·24시간 기준. 전액 환불 시 쿠폰·포인트 복구.

**Q. 통행료·주유비는?**  
임차인 부담입니다.

**Q. 사고 시?**  
즉시 고객센터 연락. 음주·무면허 시 보험 미적용.

**Q. 포인트는?**  
5,000원 이상 사용, 반납 완료 시 5% 적립, 유효 1년.
